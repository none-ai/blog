# Blog

A modern blogging platform built with Flask.

## 🎯 Features

- Create and manage blog posts
- Rich text editing
- Category management
- Tag support
- Comment system
- SEO friendly URLs
- Responsive design

## 🚀 Quick Start

```bash
# Clone the repository
git clone https://github.com/none-ai/blog.git
cd blog

# Install dependencies
pip install -r requirements.txt

# Run the application
python app.py
```

Visit http://localhost:5000

## 📋 Requirements

- Python 3.8+
- Flask
- SQLAlchemy
- See `requirements.txt` for full list

## 🛠️ Tech Stack

- Python 3
- Flask
- SQLAlchemy
- Jinja2 Templates
